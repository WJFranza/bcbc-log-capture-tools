What this project is

A real first GitHub project and the foundation for your future USB SSD interface.

GitHub starter project

This repository gives you a clean first project with an actual working script, documentation, and a public-facing GitHub Pages site.

USB SSD building block

The same tool can later be surfaced in a launcher or web menu for your technician-first bootable USB SSD.

Current and planned tools

Start simple, then keep adding modules as the system grows.

Live now

Log Capture

Collects core system information, storage details, memory and process snapshots, recent logs, networking details, and Tailscale status when available.

  • Linux-friendly
  • Creates timestamped reports
  • Good first diagnostic baseline

Getting started

Use these exact commands after you create the GitHub repository.

Run the tool locally

chmod +x scripts/log_capture.sh
./scripts/log_capture.sh

Push to GitHub

git init
git add .
git commit -m "Initial commit: BCBC log capture starter"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
git push -u origin main
